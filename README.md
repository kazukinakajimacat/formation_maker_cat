# Formation Maker

サッカーのスターティングメンバー表とフォーメーションを直感的に作れるWebアプリです。React + SVGで構築されており、作成した編成は画像として保存・共有できます。

---

## 主な機能

### 編成作成
- 8種類のフォーメーションプリセット（4-4-2 / 4-3-3 / 4-2-3-1 / 3-4-3 / 3-5-2 / 5-3-2 / 4-5-1 ほか）
- 選手をドラッグで自由に再配置可能（カスタム配置時は自動でCustomバッジ表示）
- 「配置を戻す」ボタンでプリセット位置に1タップで復元
- 控え選手のベンチ表示（1〜12人まで増減可能）
- 先発⇔ベンチ間も含めた選手の入れ替え機能

### ユニフォームカスタマイズ
- 12色のフィールド選手用カラー
- 5色のGK用カラー
- 3パターンのデザイン（無地・ストライプ・ボーダー）

### 出力
- タイトル・副題（試合名・対戦相手など）の設定
- ブラウザ内ストレージへの保存・読み込み（複数件管理）
- PNG画像としてのダウンロード（2倍解像度で書き出し）
- Web Share API経由での共有（対応OS時はネイティブシェアシートが起動）

## 技術スタック

| 領域 | 採用技術 |
|------|----------|
| フレームワーク | React 18 |
| ビルドツール | Vite |
| スタイリング | Tailwind CSS |
| アイコン | lucide-react |
| 描画 | ネイティブSVG（外部チャート/グラフィックライブラリ不使用） |
| 保存 | localStorage（ブラウザ内永続化） |
| 共有 | Web Share API（Level 2 / ファイル共有対応） |

外部APIや認証は使用していません。完全にクライアントサイドで完結します。

## セットアップ

### 必要環境
- Node.js 18 以上
- npm または pnpm / yarn

### インストール

```bash
git clone https://github.com/your-username/formation-maker.git
cd formation-maker
npm install
```

### 開発サーバー起動

```bash
npm run dev
```

### プロダクションビルド

```bash
npm run build
npm run preview
```

## 注意事項

### 実在の選手名・チーム名について
本アプリはユーザーが任意のテキストを入力できる汎用ツールです。実在の選手名・チーム名・ロゴ等のデータはバンドルされていません。

入力したテキストや作成画像をSNS等で公開する際は、肖像権・パブリシティ権・商標権等に関するご自身の責任において利用してください。営利目的での無断利用は法的リスクを伴う可能性があります。

### スクリーンショット・サンプル画像
本リポジトリに含まれるスクリーンショットやサンプル画像には、実在する選手名は使用していません。

### データの保存場所
保存した編成データはブラウザのlocalStorageに格納されます。ブラウザのキャッシュクリアや別端末への移行時は復元できません。重要なデータは画像として保存することを推奨します。
