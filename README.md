# Formation Maker

サッカーのスターティングメンバー表とフォーメーションを直感的に作れるWebアプリです。

---

## できること

### 編成作成
- 8種類のフォーメーションプリセット（4-4-2 / 4-3-3 / 4-2-3-1 / 3-4-3 / 3-5-2 / 5-3-2 / 4-5-1 ほか）
- 選手をドラッグで自由に再配置
- 控え選手のベンチ表示（1〜12人まで増減可能）
- 先発⇔ベンチを跨いだ選手の入れ替え

### ユニフォームカスタマイズ
- 12色のフィールド選手用カラー、5色のGK用カラー
- 無地・ストライプ・ボーダーの3パターン

### 保存と共有
- タイトルと副題（試合名・対戦相手など）の設定
- 作成した編成を保存・読み込み（複数件管理）
- PNG画像としてダウンロード
- スマホからはそのままSNS等に共有可能

---
